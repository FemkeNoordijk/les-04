HTML/CSS stappenplan

Om met HTML/CSS te kunnen werken heb ik een overzichtelijk stappenplan gemaakt om het wat makkelijker te maken.

Je begint met het maken van een mapje. Dit mapje voeg je toe aan je Visual Studio Code. In het mapje in VSC maak je een nieuw bestand genaamd: index.html.
Daarna maak je nog een nieuw bestand genaamd: style.css.
Deze twee bestanden moet je aan elkaar gaan koppelen dit doe je door in het html bestand een link toe te voegen. (link rel="stylesheet" href="style.css"/)
Het is verstandig om nu je map naar github te uploaden.
Nu kan je beginnen met het maken van je website in html. Dit doe je door de inhoud van je website te coderen in html.
Nu je html af is kan je op basis van html je css bestand gaan maken. Hierin kan je de inhoud van je website naar jouw smaak opmaken.
Hier kan je continue mee spelen totdat je website voldoet aan jouw eisen.
Nu kan je je website opslaan en beginnen met een nieuw project.

Succes!
